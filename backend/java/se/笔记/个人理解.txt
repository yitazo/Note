DOS命令窗口常用命令
	cd 名称\名称，切换到名称所在的路径，c:直接切换符盘，cd \直接切换到根路径
	dir，列出当前路径下所有目录
	mkdir 名称，在当前路径下创建相应文件夹目录
	del *后缀名称，名称可以模糊化，结尾带有这个名称的都会被删除
	ipconfig,查看ip地址，后面加/all可以查看更详细的信息
	ping IP地址或域名。查看两台设备能否连接。ping 域名 -t,一直ping,检查网络是否稳定，ctrl+c可以结束任意DOS命令



环境变量Path
	Path环境变量里的路径用英文分号隔开
	在DOS窗口中输入exe文件名称，将先在本路径下寻找文件，若找不到则在环境变量中寻找，都找不到则无法运行



编译和运行
	编译：javac 绝对路径（直接把文件拖进DOS窗口），或者相对路径（含有.java的路径）
	运行：java只能接字节码文件名称，只能在当前目录下寻找字节码文件，除非配置classpath



java具有健壮性
	无需手动释放空间，只需申请，自动释放。因为有自动垃圾回收机制（GC机制，底层由C++编写）



java具有可移植性
	通过JVM虚拟机屏蔽操作系统之间的差异，间接和不同的操作系统打交道（不同的操作系统有不一样的JVM虚拟机）



JDK,JVM,JRE之间的关系
	JDK：java开发工具箱
	JRE：java运行环境（运行时类库）
	JVM：java虚拟机
	JDK包含JRE，JRE包含JVM和类库。JVM无法独立安装，但其余两个可独立安装，给用户用软件只需安装JRE即可
 


类加载器
	启动java程序时，JVM将启动类加载器去寻找目标类，将目标类装载到JVM上，然后JVM启动解析器将字节码文件代码
	解析成二进制码，从而计算机能够识别指令



命名规范
	驼峰命名法，一高一低
	类名和接口名首字母大写，后面每个单词首字母大写
	变量名首字母小写，后面每个单词字母大写
	常量名每个字母大写，每个单词之间用下划线衔接



在Java中除了定义变量外不能使用逗号表达式，例如将变量的值交换时不能使用
多个赋值语句不能在定义时就使用，例如int a = b = c = 0;可以在定义完之后使用
